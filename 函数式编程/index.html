Lambda表达式：实现函数式的编程，可以避免面型对象编程之中繁琐的编程，避免结构化的繁琐 使用要求：函数中只有一个方法且为抽象方法，只能用于函数式接口 格式： ()->{} (参数,参数)->{} @FunctionalInterfa //函数式接口 @FunctionalInterfa interface Imath{ public int add(int x,int y); } public class JavaDemo{ public void static void main(){ Imath math=(t1,t2)->{ return t1+t2; }; System.out.println(math.add(10,20)); } } 当表达式中只有一行语句时可以简化如下 @FunctionalInterfa interface Imath{ public int add(int x,int y); } public class JavaDemo{ public void static void main(){ Imath math=(t1,t2)-> t1+t2 ; System.out.println(math.add(10,20)); } } 不使用lambda表达式 @FunctionalInterfa interface Imath{ public int add(int x,int y); } public class JavaDemo{ public void static void main(){ Imath math=new math(){ public int add(int x,int y){ return x+y; } }; System.out.println(math.add(10,20)); } }
